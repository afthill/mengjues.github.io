expression定义

expression可以生产（produce）数据，这个生产的过程是通过evaluation。也就是说
数据没有马上产生，是需要时才会被evaluated。

statement定义

statement是用来做事情（do things）。这里的数据会马上产生，直到最终的结果才算
当前的statement被执行完毕。
